# Portfolio

## DemoBank: Microservices with Java, SpringBoot, Docker and Kubernetes
* Domain-Driven Design (DDD) architecture
* Cloud Native Architecture
* Implemenrting Global Exception Handler
* Spring Cloud Config Server microservice 
* Spring Cloud Config Clients: Accounts, Loans and Cards microservices
* Supporting Databases H2, MySQL
* Implementing Docker
* Container Orchestration with Docker Compose
* Configuration files using GitHun Repository (encryption and decryption properties support)
* Spring Boot profiles
* Spring Cloud Bus
* Spring Cloud Config Monitoring
* Liveness and Readiness with Spring Actuator
* RabbitMQ support for reading properties inside Config Server using Actuator

See the [documentation](doc/spring_boot_cloud/spring_boot_cloud.md) for more details.<br>
See the source code [here](https://github.com/carlosjulioperez/portfolio/tree/main/microservices).
